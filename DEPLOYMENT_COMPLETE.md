## 🎨 How to Use

### Creating a Banner (Admin/Teacher):

1. **Navigate**: Dashboard Banners → "New Banner"
2. **Upload Media**:
   - Click "Upload File" for local images/videos, OR
   - Click "Enter URL" for external links
3. **Add Redirect** (optional): Enter URL to redirect users
4. **Select Audience**: Choose who should see it
5. **Set Expiry**: Default 7 days
6. **Create**: Click "Create Banner"

### Viewing Banners (All Users):

- **On Dashboard**: Automatic carousel at top of page
- **Auto-scrolls**: Every 5 seconds
- **Click banner**: Redirects to linked URL
- **Use controls**: Previous/Next buttons, Play/Pause

### Managing Banners (Admin):

- **View All**: Dashboard Banners page
- **Filter**: By audience type
- **Hide**: Temporarily hide from users
- **Delete**: Remove permanently
- **Restore**: Unhide hidden banners

## ⚠️ Important Notes

### 1. Breaking Change
Old announcements with title/message are **NOT compatible** with the new system.

**Action Required**:
- Recreate existing banners with media content
- Or archive old announcements

### 2. External URLs
When using external URLs, ensure they are:
- ✅ Publicly accessible (no login required)
- ✅ Direct links to media files
- ✅ HTTPS (secure)

**Supported Platforms**:
- Pinterest
- Instagram
- Facebook
- Ideogram
- Canva
- Google Drive (public links)
- Any public image/video URL

### 3. File Sizes
- **Images**: Maximum 10MB
- **Videos**: Maximum 50MB

### 4. Permissions
- **Create Banners**: Teachers, School Admins, Platform Admin
- **View Banners**: All users (filtered by role)
- **Hide/Delete**: Platform Admin only

## 💡 Pro Tips

1. **Use High-Quality Images**: 
   - Minimum 1200px wide for best quality
   - Use Canva or similar tools to create professional banners

2. **External URLs Work Great**:
   - Pinterest images: Right-click → "Open image in new tab" → Copy URL
   - Instagram: Use third-party tools to get direct image URLs
   - Ideogram: Generated images can be linked directly

3. **Test Your Links**:
   - Always test redirect URLs before publishing
   - Use "Open in new tab" for external sites

4. **Target Wisely**:
   - Use "Everyone" for platform-wide announcements
   - Target specific roles for relevant content

5. **Set Appropriate Expiry**:
   - Events: 1-3 days before
   - Promotions: 7-14 days
   - General info: 30+ days
