🚀 HOW TO USE

CREATE BANNER:
1. Go to "Dashboard Banners" → "New Banner"
2. Upload image/video OR paste external URL
3. Add redirect URL (optional)
4. Select target audience
5. Click "Create Banner"

VIEW BANNERS:
• Dashboards: Auto-scrolling carousel at top
• Management: Grid view with filters

MANAGE BANNERS (Admin):
• Hide: Temporarily hide from users
• Delete: Remove permanently
• Filter: By audience type

⚠️ IMPORTANT NOTES

1. BREAKING CHANGE:
   Old announcements with title/message are NOT compatible.
   You'll need to recreate banners with media content.

2. EXTERNAL URLS:
   Make sure URLs are:
   • Publicly accessible (no login)
   • Direct links to media files
   • HTTPS (secure)

3. FILE SIZES:
   • Images: Max 10MB
   • Videos: Max 50MB

4. PERMISSIONS:
   • Create: Teachers, Admins
   • Hide/Delete: Platform Admin only
   • View: All users (filtered by role)
