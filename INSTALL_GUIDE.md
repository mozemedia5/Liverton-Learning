# Liverton Learning - PWA Installation Guide

## For End Users

### Windows & Mac (Desktop)

#### Chrome / Edge / Brave
1. Visit the Liverton Learning website
2. Look for the **"Download"** button in the bottom-right corner
3. Click it and select "Install"
4. The app will appear on your desktop and taskbar

**Alternative (Chrome/Edge):**
- Click the install icon in the address bar
- Or: Right-click the tab ‚Üí "Install app"

#### Firefox
1. Visit the Liverton Learning website
2. Look for install prompt in the address bar
3. Click "Add" to install

## Troubleshooting

### "Download button not showing" (Chrome)
- **Solution:** 
  - Refresh the page
  - Make sure you're not already installed (check home screen)
  - Try clearing cache: Settings ‚Üí Privacy ‚Üí Clear browsing data
  - Make sure you're using a recent version of Chrome

### "Install option not available" (Firefox)
- **Solution:**
  - Refresh the page
  - Firefox requires service worker to be registered
  - Try a fresh browser window

### "App won't load offline"
- **Solution:**
  - Make sure you've visited the app while online first
  - Service worker needs to cache content
  - Try force-refresh (Ctrl+Shift+R on Windows, Cmd+Shift+R on Mac)

### "App not appearing on home screen"
- **Solution:**
  - On Android: Restart phone and check
  - On iOS: Make sure Safari is up to date
  - On Desktop: Check applications menu if not in taskbar
